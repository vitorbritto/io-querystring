# I/O Query String

Extract and return a querystring from a given URL.


## Requires

- [Node.JS](http://nodejs.org/)


## Usage

```javascript
ioQuery.parse(window.location.href);
```


## Includes

- [Mocha](http://visionmedia.github.io/mocha/)
- [Chai](http://chaijs.com/)
